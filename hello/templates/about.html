About This Application

This is a simple Student Management Application built using Django and Bootstrap.

The application allows users to easily add, update, and delete student records. It demonstrates core Django features such as models, views, templates, forms, and messages.

The goal of this project is to provide a basic understanding of how CRUD (Create, Read, Update, Delete) operations work in Django with a clean and responsive interface.
